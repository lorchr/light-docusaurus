  • PVE 和 Esxi性能对比

申请证书​

在 VMware 网站中，我们能够找到 ESXi 8 和 ESXi 7 的使用授权的申请地址：

  • VMware vSphere Hypervisor 8
  • VMware vSphere Hypervisor 7
  1. 注册登录账号
  2. 跳转申请地址，填写表单（individual），等待审核通过

一般会立即通过并显示证书及下载链接

编译镜像​

注意： 8.0版本开始，VMWare删减了一些硬盘 网卡的驱动，可能导致网卡识别或硬盘识别异常，需要自己用离线包打驱动。具体操作可以参考快速构建和安装干净的 ESXi 8 镜像指南 或 快速构建和安装干净的 ESXi 8 镜像指南

  1. 在证书页面下载官方ISO镜像 及 离线Zip安装包
  2. 从VMware兼容性指南 VMware Compatibility Guide查询硬件支持情况，包括CPU 网卡 PCIE SATA NVMe
  3. 如果缺少某个驱动，可以从flings ESXi 社区网络驱动程序下载
    1. ESXi 的 PCIe 社区网络驱动程序：community-networking-driver-for-esxi
    2. ESXi 的 USB 社区网络驱动程序：usb-network-native-driver-for-esxi
    3. ESXi 的 NVMe 社区驱动程序：community-nvme-driver-for-esxi
  4. 编译镜像，添加驱动，从这里下载

注意 由于VMware决定关闭 `flings.vmware.com`` 网站，对相关内容做了迁移

  1. 网站跳转到Code Samples and PowerCLI Example Scripts
  2. Esxi for ARM 迁移到了ESXi for ARM
  3. 相关文档迁移到了ESXi-Arm Fling Documents - VMware ESXi-Arm Documentation
  4. 部分内容（包括驱动）可以在这里找到

安装Esxi​

  1. 将镜像复制到Ventoy制作的U盘中，或者使用Rufus烧录到U盘
  2. 将U盘插入到主机上并按 Del(或Esc, F2, F7, F12)进入到BIOS，再Boot中将U盘设置为第一启动项
  3. 选择Esxi镜像进行安装
  4. 在安装页面显示EFI/boot/boot.cfg时，按Shift + o进入设置，输入cdromBoot runweasel autoPartitionOSDataSize=8192
